Accounts & creds (HQ): OF/Fansly logins, social handles, credential records.
Brand survey (60+ Q): pricing, dos/don'ts, prohibited words, voice, pet-names → for the chat team.
Back-end provisioning (the orphaned run-setup): Dropbox folder tree, 2 file-request links, HQ account rows, SM Setup Request.
Social go-live: default "Palm IG" rows (the broken bit), 3 IG usernames, Telegram IG + FB Topic IDs (Grid Planner errors without both), Social Media Editing toggle.
Content intelligence: profile docs → Run Analysis → tags/voice/dos-don'ts, content pillars, inspo handles, tag weights, Weekly Reel Quota, Music DNA.
AI (if enabled): AI ref images (front/back/face) + clone approval, Kling element, persona profile + warmup + vault IDs.
Publishing: Publer sync → map account → AI consent on file.
Finance: Revenue Account named to match the Google-Sheet tab exactly, management start date, whale Telegram topic.
